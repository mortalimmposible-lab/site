Задачи на адресацию (URL, email, IP)

6 реальных заданий из ОГЭ. Восстановите последовательность.

Задача 1
Доступ к файлу tiger.doc, сервер zoo.org, протокол http.
Фрагменты: А) .doc ; Б) zoo ; В) // ; Г) :// ; Д) tiger ; Е) .org ; Ж) http.
Запишите последовательность букв (порядок фрагментов).
Решение: Адрес: протокол + :// + сервер + / + файл. http (Ж) + :// (Г) + zoo (Б) + .org (Е) + / (В) + tiger (Д) + .doc (А). Получаем ЖГБЕВАД.
Ответ: ЖГБЕВАД.
Задача 2
Доступ к файлу start.exe, сервер game.com, протокол http.
Фрагменты: А) start ; Б) / ; В) .exe ; Г) http ; Д) game ; Е) .com ; Ж) :// .
Запишите последовательность букв.
Решение: http (Г) + :// (Ж) + game (Д) + .com (Е) + / (Б) + start (А) + .exe (В). → ГЖДЕБАВ.
Ответ: ГЖДЕБАВ.
Задача 3
Почтовый ящик wait_for_mail на сервере GorodN.ru.
Фрагменты: А) GorodN ; Б) mail ; В) for_ ; Г) .ru ; Д) wait_ ; Е) @ .
Последовательность букв для адреса.
Решение: Адрес: имя_ящика + @ + сервер. wait_ (Д) + for_ (В) + mail (Б) → wait_for_mail → + @ (Е) + GorodN (А) + .ru (Г) = ДВБЕАГ.
Ответ: ДВБЕАГ.
Задача 4
Ящик teacher на сервере shkola1.mos.ru. Фрагменты: 1) @ ; 2) .ru ; 3) shkola1 ; 4) . ; 5) teacher ; 6) mos.
Последовательность цифр.
Решение: teacher (5) + @ (1) + shkola1 (3) + . (4) + mos (6) + .ru (2) → 5 1 3 4 6 2. Напомним, как формируется почтовый адрес в сети Интернет. Сначала указывается имя почтового ящика, потом «@», потом сервер, на котором находится почтовый ящик. Таким образом, искомый адрес будет следующим: teacher@shkola1.mos.ru.   Ответ: 513462.
Ответ: 513462.
Задача 5
Фрагменты IP-адреса: А) 2.12 ; Б) 22 ; В) .30 ; Г) 5.121 .
Восстановите IP-адрес (например, 5.121...). Последовательность букв.
Решение: IP-адрес представляет собой четыре разделенных точками числа, каждое из которых не больше 255. Под буквой В указано «.30». Так как числа в IP-адресе не могут быть больше 255, нельзя добавить в конце этого числа еще один разряд. Следовательно, этот фрагмент  — последний. Рассмотрим фрагмент под буквой Б. Так как числа в IP-адресе не могут быть больше 255, фрагмент Б должен быть на первом месте. В конце фрагмента Г  — число 121, отделенное точкой. Так как в IP-адресе не может быть числа, большего 255, то за фрагментом Г должен следовать фрагмент, начинающийся с точки. Значит, фрагмент Г идет перед фрагментом В. Следовательно, ответ БАГВ.
Задача 6
Фрагменты: А) 17 ; Б) .44 ; В) 4.144 ; Г) 9.13 .
Восстановите IP-адрес. Порядок букв.
Решение:IP-адрес представляет собой четыре разделенных точками числа, каждое из которых не больше 255. Под буквой Б указано «.44». Так как числа в IP-адресе не могут быть больше 255, нельзя добавить в конце этого числа еще один разряд. Следовательно, этот фрагмент  — последний. В конце фрагмента В  — число 144, отделенное точкой. Так как в IP-адресе не может быть числа, большего 255, то за фрагментом В должен следовать фрагмент, начинающийся с точки. Значит, фрагмент В идет перед фрагментом Б. Фрагмент под буквой Г не может быть первым, поскольку в таком случае за ним вынужден стоять фрагмент А, а в IP-адресе не может быть числа, большего 255. Следовательно, ответ АГВБ.
