Задание 3: Логика

Основные логические операции

  • НЕ (Инверсия): Переворачивает смысл. Если условие X истинно, то НЕ X — ложно. По сути, НЕ (X > 5) превращается в X ≤ 5.
  • И (Конъюнкция): Выражение истинно, только если оба условия верны одновременно.
  • ИЛИ (Дизъюнкция): Выражение истинно, если верен хотя бы один из вариантов.

Алгоритм решения

  1. Избавьтесь от «НЕ»: Раскройте скобки, поменяв знаки (например, НЕ (X > 5) → X ≤ 5, НЕ (X < 10) → X ≥ 10).
  2. Выпишите условия: Составьте список ограничений для X.
  3. Найдите ответ: Подберите число, которое подходит под все условия одновременно (если стоит «И») или хотя бы одно (если «ИЛИ»).

Пример

Найдите наименьшее целое число X, для которого истинно выражение: (X > 16) И НЕ (X нечётное).

Решение по алгоритму:
1. Убираем НЕ: НЕ (X нечётное) = X чётное.
2. Условия: X > 16 и X чётное.
3. Первое чётное число больше 16 — это 18.
Ответ: 18.
