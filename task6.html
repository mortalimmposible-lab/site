Задачи на анализ программ

Определите, сколько раз программа напечатает «YES».

Задача 1
Условие: if s > 5 and t > 5: print("YES")
Пары чисел:
(6,8); (3,5); (-7,2); (7,7); (9,8); (-1,3); (-4,5); (6,9); (2,-1)
Сколько раз программа напечатает «YES»?
Решение: Условие истинно, если оба числа >5.
Проверяем: (6,8) → да; (3,5) → нет; (-7,2) → нет; (7,7) → да; (9,8) → да; (-1,3) → нет; (-4,5) → нет; (6,9) → да; (2,-1) → нет. Итого 4? Пересчитаем: (6,8), (7,7), (9,8), (6,9) → 4. Но в условии сказано (6,8); (7,7); (9,8); (6,9) — 4 раза. Ответ: 4.
Ответ: 4.
Задача 2
Условие: if s > 10 and t > 10: print("YES")
Пары: (10,10); (10,6); (-4,8); (2,9); (12,7); (-11,4); (-8,13); (10,9); (11,11).
Сколько раз напечатает «YES»?
Решение: Оба числа должны быть строго больше 10.
Проверяем: (10,10) → нет; (10,6) → нет; (-4,8) → нет; (2,9) → нет; (12,7) → нет; (-11,4) → нет; (-8,13) → нет; (10,9) → нет; (11,11) → да (11>10 и 11>10). Только одна пара.
Ответ: 1.
Задача 3
Условие: if s > 10 or t > 10: print("YES")
Пары: (1,2); (11,2); (1,12); (11,12); (-11,-12); (-11,12); (-12,11); (10,10); (10,5).
Сколько раз напечатает «YES»?
Решение: Условие истинно, если хотя бы одно число >10.
Проверяем: (1,2) → нет; (11,2) → да; (1,12) → да; (11,12) → да; (-11,-12) → нет; (-11,12) → да; (-12,11) → да; (10,10) → нет; (10,5) → нет. Итого: 5 раз.
Ответ: 5.
Задача 4
Условие: if s > 6 and t < 7: print("YES")
Пары: (7,3); (2,7); (6,10); (5,3); (5,4); (-11,4); (-8,9); (7,3); (9,1).
Сколько раз напечатает «YES»?
Решение: Условие: s > 6 и t < 7.
Проверяем: (7,3) → 7>6 и 3<7 → да; (2,7) → 2>6? нет; (6,10) → 6>6? нет; (5,3) → 5>6? нет; (5,4) → нет; (-11,4) → нет; (-8,9) → нет; (7,3) → да (повтор); (9,1) → 9>6 и 1<7 → да. Итого: (7,3) дважды и (9,1) → 3 раза.
Ответ: 3.
Задача 5
Условие: if s < 9 or t < 9: print("YES")
Пары: (9,9); (9,10); (8,5); (11,6); (-11,10); (-5,9); (-10,10); (4,5); (8,6).
Сколько раз напечатает «YES»?
Решение: Условие истинно, если хотя бы одно число <9.
Проверяем: (9,9) → оба >=9 → нет; (9,10) → оба >=9 → нет; (8,5) → 8<9 → да; (11,6) → 6<9 → да; (-11,10) → -11<9 → да; (-5,9) → -5<9 → да; (-10,10) → -10<9 → да; (4,5) → оба <9 → да; (8,6) → оба <9 → да. Итого: 7 раз.
Ответ: 7.
