Задание 9: Подсчёт количества путей в графе

Пошаговый алгоритм решения

  1. Точка старта: Возле начального города (например, А) ставим цифру 1. Это значит, что в пункт А есть 1 способ попасть (мы уже там).
  2. Движение по графу: Идём к следующим городам по стрелкам. Количество путей в текущий город равно сумме цифр во всех городах, из которых в него ведут стрелки.
  3. Правило очередности: Нельзя считать город, если ещё не посчитаны все города, из которых в него ведут дороги.
    Пример: Если в город В ведут стрелки из А и Б, сначала нужно узнать значения и для А, и для Б.
  4. Финальная точка: Число, которое получится в конечном пункте, и будет ответом.

Если есть дополнительные условия

«Путь проходит через город K»:
Сначала удалите (зачеркните) все дороги, которые ведут в обход города K.
Посчитайте пути только по оставшимся стрелкам, которые гарантированно ведут в K, а затем из него.

«Путь НЕ проходит через город K»:
Просто зачеркните сам город K и все стрелки, которые в него входят или из него выходят.
Посчитайте количество путей по оставшейся схеме.

Пример

Граф: A→B, A→C, B→D, C→D. Найдите количество путей из A в D.

Решение по алгоритму:
1. В А ставим 1.
2. В B ведёт только из A → B = 1.
3. В C ведёт только из A → C = 1.
4. В D ведут стрелки из B (1) и из C (1) → D = 1+1 = 2.
Ответ: 2.
