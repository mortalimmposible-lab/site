Задание 7: Адресация (URL)

Формула (с каталогом)

Протокол :// Сервер / Каталог / Файл

Если каталогов несколько, они просто идут друг за другом через косую черту (/):

Протокол://Сервер/Папка1/Папка2/Файл
  • Протокол — http, https, ftp и т.д.
  • Сервер — доменное имя (например, obr.ru, files.net).
  • Каталог(и) — папки на сервере (могут отсутствовать или быть вложенными).
  • Файл — конечный документ с расширением (.html, .mp3, .docx и т.д.).

Пример

Составьте URL из частей: ftp, net.ru, music, mp3, song.mp3 (где music — папка, mp3 — подпапка, song.mp3 — файл).

Решение:
Протокол: ftp
Сервер: net.ru
Каталоги: music/mp3
Файл: song.mp3
Ответ: ftp://net.ru/music/mp3/song.mp3
