Задачи на логические выражения

6 реальных заданий из ОГЭ. Проверьте свои знания.

Задача 1
Напишите количество натуральных чисел, для которых истинно высказывание:
НЕ (Число > 15) И НЕ (Число четное).
Решение:
Логическое «И» истинно только тогда, когда истинны оба высказывания. Запишем выражение в виде (X <= 15) И (Число нечетное). Выпишем подходящие натуральные числа: 1, 3, 5, 7, 9, 11, 13 и 15. Их количество равно 8.Ответ: 8
Задача 2
Дано четыре числа: 638, 442, 357, 123. Для какого из них истинно:
НЕ (Первая цифра четная) И (Сумма цифр четная).
Решение:
Первая цифра четная: у 638 (6), 442 (4) → нет. У 357(3) и 123(1) – нечётные, условие НЕ (первая цифра четная) истинно. Теперь сумма цифр четная: 357: 3+5+7=15 (нечет), 123: 1+2+3=6 (чет). Подходит 123.
Ответ: 123.
Задача 3
Напишите наибольшее натуральное число x, для которого истинно:
(x < 5) ИЛИ (x > 3).
Решение:
Логическое «ИЛИ» истинно во всех случаях, кроме случая, когда ложны оба высказывания. Запишем выражение в виде (x < 5) ИЛИ (x <= 3). Значит, наибольшее число, для которого высказывание будет истинным  — 4.Ответ: 4
Задача 4
Напишите наибольшее целое число x, для которого истинно:
(X >= 6) И НЕ (X > 12).
Решение:
НЕ (X > 12) = X ≤ 12. Условие: X ≥ 6 и X ≤ 12. Наибольшее целое — 12.
Ответ: 12.
Задача 5
Напишите наибольшее целое число x, для которого истинно:
НЕ (X < 4) ИЛИ НЕ (X >= 9).
Решение:
Запишем выражение в виде (X >= 4) И (X < 9). Значит, наибольшее число, для которого высказывание будет истинным  — 8.Ответ: 8.
Задача 6
Напишите наибольшее целое число x, для которого истинно:
НЕ (X <= 11) ИЛИ НЕ (X >= 17) ИЛИ (X нечетное).
Решение:
Запишем выражение в виде (X > 11) И (X < 17) И (X нечетное). Значит, наибольшее число, для которого высказывание будет истинным  — 15. А наименьшее число, для которого высказывание будет истинным  — 13. Ответ: 15.
