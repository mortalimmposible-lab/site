Задание 8: Поисковые запросы (круги Эйлера)

1. Главная формула

|A ∪ B| = |A| + |B| – |A ∩ B|
  • |A| — количество страниц по первому слову.
  • |B| — количество страниц по второму слову.
  • |A ∩ B| — запрос с «И» (пересечение, обычно самое маленькое число).
  • |A ∪ B| — запрос с «ИЛИ» (объединение, самое большое число).

Тебе всегда даны три значения из четырех. Просто подставь их в формулу и найди неизвестное.

2. Лайфхак «Сложи бока — вычти лишнее»

Если лень запоминать формулу, запомни логику:

  • Сложи одиночные запросы (|A| + |B|).
  • Вычти из них либо |A ∩ B|, либо |A ∪ B| (то, что дано в условии).
  • Получишь оставшуюся часть.

Пример

Дано: Орёл = 500, Решка = 400, Орёл & Решка = 100. Найдите количество страниц по запросу Орёл | Решка.

Решение по формуле:
|Орёл ∪ Решка| = |Орёл| + |Решка| – |Орёл ∩ Решка| = 500 + 400 – 100 = 800.
Лайфхак: складываем бока (500+400=900), вычитаем лишнее (100) → 800.
Ответ: 800.
