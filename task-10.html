Задание 10: Системы счисления

Как переводить в десятичную систему

Нужно расписать число по разрядам (справа налево, начиная с 0) и умножить каждую цифру на основание системы в соответствующей степени.

Примеры перевода:

Двоичная (основание 2):
1011₂ = 1·2³ + 0·2² + 1·2¹ + 1·2⁰ = 8 + 0 + 2 + 1 = 11₁₀
Восьмеричная (основание 8):
257₈ = 2·8² + 5·8¹ + 7·8⁰ = 2·64 + 5·8 + 7·1 = 128 + 40 + 7 = 175₁₀
Шестнадцатеричная (основание 16):
Помним: A=10, B=11, C=12, D=13, E=14, F=15.
1F₁₆ = 1·16¹ + 15·16⁰ = 16 + 15 = 31₁₀
A3₁₆ = 10·16¹ + 3·16⁰ = 160 + 3 = 163₁₀

Пример для самостоятельного решения

Сравните числа 1011₂ и 1B₁₆. Какое из них больше?

Решение:
1) Переводим 1011₂ в десятичную: 1·2³ + 0·2² + 1·2¹ + 1·2⁰ = 8 + 0 + 2 + 1 = 11₁₀.
2) Переводим 1B₁₆ в десятичную: B = 11, значит 1·16¹ + 11·16⁰ = 16 + 11 = 27₁₀.
3) Сравниваем: 27 > 11, значит 1B₁₆ больше.
Ответ: 1B₁₆ больше.
