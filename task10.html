Задачи на системы счисления

Перевод чисел, арифметика, количество единиц.

Задача 1
Переведите число 90 из десятичной системы счисления в двоичную. Сколько единиц содержит полученное число? В ответе укажите одно число — количество единиц.
Решение: 90₁₀ = 1011010₂. Считаем единицы: 1+0+1+1+0+1+0 = 4.
Ответ: 4.
Задача 2
Переведите число 111 из десятичной системы счисления в двоичную. Сколько единиц содержит полученное число? В ответе укажите одно число — количество единиц.
Решение: 111₁₀ = 1101111₂. Считаем единицы: 6.
Ответ: 6.
Задача 3
Переведите двоичное число 1010101 в десятичную систему счисления.
Решение: 1·2⁶ + 0·2⁵ + 1·2⁴ + 0·2³ + 1·2² + 0·2¹ + 1·2⁰ = 64 + 0 + 16 + 0 + 4 + 0 + 1 = 85.
Ответ: 85.
Задача 4
Некоторое число в двоичной системе счисления записывается как 101111. Запишите это число в десятичной системе.
Решение: 1·32 + 0·16 + 1·8 + 1·4 + 1·2 + 1·1 = 32 + 0 + 8 + 4 + 2 + 1 = 47.
Ответ: 47.
Задача 5
Некоторое число в двоичной системе счисления записывается как 11010010. Запишите это число в десятичной системе.
Решение: 1·2⁷ + 1·2⁶ + 0·2⁵ + 1·2⁴ + 0·2³ + 0·2² + 1·2¹ + 0·2⁰ = 128 + 64 + 0 + 16 + 0 + 0 + 2 + 0 = 210.
Ответ: 210.
Задача 6
Вычислите значение арифметического выражения: 11011111₂ + 1011₈ - 111₁₆. В ответе запишите десятичное число.
Решение: 11011111₂ = 223, 1011₈ = 521, 111₁₆ = 273; 223 + 521 – 273 = 471.
Ответ: 471.
